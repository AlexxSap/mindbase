#+title: Работа с тудушками

Чтобы сделать тудушку нужно поставить * и написать TODO
#+begin_src
 * TODO some text
#+end_src

Выбрать состояние можно с помощью комбинации C-c C-t или S-left / S-right
Собрать дерево тудушек можно командой org-sparse-tree
Вставить тудушку можно командой org-insert-todo-heading или S-M-RET

Можно установить зависимость от вложенности, поместив тудушки ниже уровнем
#+begin_src
 * TODO Blocked until (two) is done
 ** DONE one
 ** IDEA two
#+end_src

Так же можно установить приоритет тудушки с помощью S-up или S-down.
Можно создавать чек-листы с помощью [ ], [-], [x].
Положение вложенности определяет зависимость.
#+begin_src
- [ ] call people
  - [ ] Peter
  - [ ] Sarah
#+end_src
Поменять состояние чекбокса можно C-c C-c
