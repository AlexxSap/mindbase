**Как создать простой проект**
В каталоге проектов набираем (name это имя проекта)
```shell
stack new name
```

Для создания проекта можно использовать схему __simple__ - она создаёт меньше файлов в проекте.
```shell
stack new name simple
```

**Сборка**
```shell
stack build
```

После этого внутри каталога __.stack-work__ появиться исполняемый файл, его можно запустить так
```shell
stack exec name
```

**Форматировать (отсупы и всякое такое) файл**
```shell
fourmolu -i Module.hs
```
Установить форматтер можно командой:
```shell
stack install fourmolu
```

**Форматировать каталог**
```shell
fourmolu -i src
```
